# Popular List Methods

## Basics

```
list = []
list[x]
list[x:y]
```

## Methods

| Method        | Description           |
| ------------- | --------------------- |
| append(x)     | Add at end            |
| insert(i,x)   | Insert at index       |
| extend(list2) | Add multiple elements |
| remove(x)     | Remove value          |
| pop()         | Remove last           |
| pop(i)        | Remove index          |
| clear()       | Remove all            |
| sort()        | Sort list             |
| reverse()     | Reverse list          |
| count(x)      | Count value           |
| index(x)      | Find index            |

## Built-in Functions

```
min(list)
max(list)
sum(list)
len(list)
```

Lists are **mutable**.
